Typing Speed Test

RULES

  1. To start the test, click on the text box below and start typing the text displayed above it.

  2. You will be given a paragraph to type. The paragraph will be displayed when you start the test.

  3. The timer will start automatically as soon as you start typing.

  4. You can use backspace to correct your mistakes.

  5. If you make a mistake while typing, the timer will not stop. You will have to type the text correctly to stop the timer.
When you finish typing the text correctly, the time it took you will be displayed.
